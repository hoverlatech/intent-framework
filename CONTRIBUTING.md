# Contributing to INTENT Framework

Thank you for your interest in contributing. This repo is the public home of the INTENT Framework, a governance operating model for AI-native product development.

## What We Welcome

- **Template improvements:** Better defaults, clearer placeholder text, additional sections
- **Concept doc expansions:** Deeper explanations, additional examples, clearer diagrams
- **Real-world examples:** Worked examples from real or realistic projects
- **Tool integration guides:** How INTENT works with specific tools (Claude Code, Cursor, Copilot, Kiro, etc.)
- **Translations:** Localized versions of templates or docs
- **Bug fixes:** Typos, broken links, formatting issues

## How to Contribute

1. **Fork** this repository
2. **Create a branch** from `main` (e.g., `improve-quickstart-guide`)
3. **Make your changes** following the style guidelines below
4. **Submit a pull request** with a clear description of what you changed and why

## Style Guidelines

- **Audience:** Write for engineering managers, TPMs, and senior engineers
- **Tone:** Direct, practical, confident. Not academic, not salesy.
- **Structure:** Use tables for structured data. Use prose for explanations. Keep paragraphs short.
- **Templates:** Keep templates tool-agnostic. They should work with any AI coding tool or methodology.
- **Formatting:** Use standard Markdown. Do not use em dashes. Use commas, periods, or parentheses instead.

## What to Avoid

- Marketing language or hype
- Tool-specific instructions inside templates (put those in guides instead)
- Changes that break backward compatibility with existing template users
- Large restructuring without prior discussion in an issue

## Code of Conduct

Be respectful. Be constructive. Assume good intent. We are building something useful together, and every contributor deserves a professional, welcoming environment.

If you experience or witness behavior that violates this principle, contact contact@hoverlatech.com.

## Questions?

Open an issue for questions about the framework, suggestions for improvement, or to discuss larger changes before starting work.
